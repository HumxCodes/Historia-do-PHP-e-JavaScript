História e Evolução do PHP e JavaScript

O PHP (Hypertext Preprocessor) é uma linguagem de programação de código aberto amplamente usada para desenvolvimento web. Ela é especialmente adequada para a criação de sites e aplicativos da web dinâmicos, nos quais o conteúdo é gerado a partir de um banco de dados ou em resposta a ações do usuário.


O PHP foi criado por Rasmus Lerdorf em 1994, embora a versão original fosse uma coleção de scripts Perl que ele havia escrito para rastrear acessos ao seu currículo online. Ele chamou esse conjunto de scripts de "Personal Home Page Tools."



Origem PHP (1995)

A primeira versão oficial Criada por Rasmus Lerdorf do PHP era uma linguagem simples de script para a criação de páginas web dinâmicas.


PHP 2.0 (1997)

PHP/FI (Form Interpreter) é lançado como uma reescrita do PHP por Zeev Suraski e Andi Gutmans. Esta versão inclui um analisador HTML embutido. O PHP 2 é lançado como parte do PHP/FI.


PHP 3.0 (1998)

O PHP 3 é lançado, marcando a primeira versão amplamente usada do PHP. Neste ponto, o PHP é oficialmente referido como "PHP: Hypertext Preprocessor". Introdução de suporte a bancos de dados como MySQL e PostgreSQL.


PHP 4.0 (2000)

O PHP 4 é lançado com várias melhorias de desempenho e recursos, incluindo a nova Zend Engine. Este é um marco importante para a linguagem.


PHP 5.0 - 5.6 (2004 - 2014)

2004: O PHP 5 é lançado, com uma grande atualização na forma de recursos de orientação a objetos, incluindo a introdução das palavras-chave public, private e protected.

2009: O PHP 5.3 é lançado com recursos como namespaces e closures, tornando o PHP mais orientado a objetos e funcional.

2012: O PHP 5.4 é lançado com recursos como traits, short array syntax ([]), e o servidor web embutido. O uso de register_globals é removido.

2013: O PHP 5.5 é lançado, Introduzindo recursos como geradores, a palavra-chave finally para blocos try-catch, e suporte a funções variadicas. Também melhorou o desempenho e a segurança.

2014: O PHP 5.6 é lançado, introduzindo a sintaxe de constante de matriz (const FOO = [1, 2, 3]) e a função ... para argumentos variáveis.


PHP 7.0 - 7.4 (2015 - 2020)

2015: O PHP 7 é lançado, trazendo melhorias significativas de desempenho e recursos, incluindo o operador de nave espacial (<=>) para comparações e o tipo de declaração de retorno.

2017: O PHP 7.1 é lançado, introduzindo tipos de retorno mais precisos, suporte a valores nulos em tipos, e a palavra-chave void para funções que não retornam valor.

2018: O PHP 7.2 é lançado, com melhorias de desempenho, suporte a espaços de nomes aninhados, e a função object para criar objetos anonimamente.

2019: O PHP 7.3 é lançado, trazendo funcionalidades como suporte a sintaxe de anotação de tipos de argumentos, is_countable, e array_key_first/array_key_last.

2020: O PHP 7.4 é lançado, introduzindo recursos como propriedades tipadas, a atribuição de coalescência nula (??), e funções de setas.

PHP 8.0 - 8.3 (2020 - 2023)

2020: O PHP 8 é lançado, Introduzindo recursos como parâmetros nomeados e parâmetros variadicos em classes e interfaces, expandindo a flexibilidade e a expressividade do código orientado a objetos.

2021: O PHP 8.1 é lançado trouxe aprimoramentos adicionais, incluindo aprimoramentos na manipulação de tipos, operadores de atribuição compostos e mais recursos.

2022: O PHP 8.2 é lançadon, Incluindo muitos novos recursos e otimizações. Classe somente leitura, tipo independente, null, false e true como tipos stand alone, propriedades dinâmicas obsoletas, melhorias de desempenho e muito mais.

2023: O PHP 8.3 está previsto para ser lançando em 23/11/2023.


O JavaScript é uma linguagem de programação de alto nível amplamente usada para desenvolvimento web. Ela permite tornar as páginas web interativas, possibilitando a criação de elementos dinâmicos e a comunicação com o navegador do usuário.


O JavaScript foi criado por Brendan Eich enquanto trabalhava na Netscape Communications Corporation. A primeira versão do JavaScript foi lançada em 1995. A história da linguagem JavaScript está intimamente ligada à história dos navegadores da web e à necessidade de tornar as páginas da web mais interativas.



Origem JavaScript (1995)

A versão original do JavaScript (1.0) desenvolvida por Brendan Eich na Netscape, inicialmente chamado de "Mocha" e depois "LiveScript" antes de se tornar JavaScript.


JavaScript e ECMA (1996 - 1999)

1996: Submissão do JavaScript à ECMA International para padronização, levando à criação do padrão ECMAScript.

1997: Lançamento da primeira versão padronizada, ECMAScript 1.

1999: Lançamento do ECMAScript 3, que trouxe melhorias significativas, como tratamento de exceções aprimorado e suporte a regex.


ECMAScript 5 (2009)

Lançamento do ECMAScript 5, introduzindo novos recursos como getters e setters, métodos de array aprimorados e suporte integrado para JSON.


ECMAScript 6 (2015)

Lançamento do ECMAScript 6 (também conhecido como ES6), uma grande atualização que trouxe recursos como classes, let e const, setas, módulos, e muito mais. Essa versão teve um impacto significativo no desenvolvimento web.


ECMAScript 7 - 10 (2016 - 2019)

Apartir do ES7 houve a adoção de um ciclo de lançamento anual para o ECMAScript, o que permitiu a adição contínua de novos recursos e melhorias de forma consistente.


ECMAScript 11 (2020)

Lançamento do ECMAScript 11 (ES11), que introduziu o operador de coalescência nula (??) para tratamento seguro de valores nulos.


ECMAScript 12 (2021)

Lançamento do ECMAScript 12 (ES12), trazendo recursos adicionais como Record e Tuple.


ECMAScript 13 (2022)

O ECMAScript 13 (ES12), Introduziu várias melhorias, como declarações de campos de classe, métodos e campos privados, o operador await no nível superior, a propriedade .cause para erros e os métodos .findLast e .findLastIndex para encontrar o último item em arrays.
